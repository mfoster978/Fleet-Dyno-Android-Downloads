# Fleet Dyno for Android

Download the newest signed Fleet Dyno Android installer:

## [Download the latest Fleet Dyno APK](https://github.com/mfoster978/Fleet-Dyno-Android-Downloads/releases/latest/download/FleetDyno-latest.apk)

This public repository always points to the newest APK built from the private Fleet Dyno source repository. Each successful Android build replaces the previous `android-latest` release, and the release page includes automatically generated notes describing the user-visible changes.

### Installing

1. Open the download link on your Android device.
2. If Android asks, allow your browser or file manager to install apps from this source.
3. Open the downloaded APK and follow the installation prompts.

Only download Fleet Dyno from this repository or Google Play. The Android source code, signing key, and deployment credentials are not stored here.
