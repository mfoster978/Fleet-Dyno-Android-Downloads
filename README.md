## Welcome to Fleet Dyno

Fleet Dyno brings connected fleet operations into one secure mobile experience for drivers, dispatchers, fleet managers, owner-operators, administrators, and authorized customers. The Android app works alongside the Fleet Dyno web portal so users can stay connected from the road, office, or any supported browser.

Depending on your account permissions and configured integrations, Fleet Dyno supports:

- Dispatching, active loads, trip sheets, stops, and operational updates
- Live fleet location, vehicle status, route context, and shared tracking
- Multi-camera vehicle views, playback, event review, and authorized downloads
- Driver, truck, customer, and fleet-management workflows
- Payroll statements, account ledgers, invoices, receipts, fuel, and documents
- Secure electronic documents, signatures, completion records, and audit trails
- Training, notifications, support requests, and account management

Access to individual features is based on your organization, assigned role, and administrator settings.

## Installing without Google Play

1. Tap **Download Latest Signed APK** above from your Android phone or tablet.
2. Open the downloaded `FleetDyno-latest.apk` file.
3. If Android blocks the installation, follow its prompt to allow installs from that browser or file manager.
4. Return to the APK and tap **Install**.
5. Open Fleet Dyno and sign in with your authorized account.

Android may display an “unknown apps” warning because the file is being installed directly instead of through Google Play. Only approve the installation when the address begins with:

```text
https://github.com/mfoster978/Fleet-Dyno-Android-Downloads/
```

## Always the newest direct edition

The download button uses a permanent address. When a new Android build is completed successfully:

- A newly signed APK is created from the private Fleet Dyno source repository.
- Automated release notes summarize the user-visible changes.
- The previous `android-latest` release and APK are replaced.
- This page continues pointing to the newest available APK without changing its link.

Google Play internal testing remains a separate release channel and is updated only when the Android build number increases.

## Updating an existing direct installation

Download the latest APK and install it over your current direct installation. Android should treat it as an update when the installed package and signing identity match.

> [!CAUTION]
> Switching between the Google Play edition and this direct-download edition can sometimes produce an Android signature-mismatch warning. If Android requires you to uninstall first, uninstalling may remove locally stored app data. Confirm that important information is synchronized before uninstalling, and use one installation channel consistently whenever possible.
